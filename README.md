# Image Editor Web App

A simple and interactive **Image Editor** built using **HTML, CSS, and JavaScript**.  
This app allows users to upload an image, apply various filters, use presets, reset changes, and download the edited image.

---

## 🚀 Features

- Upload image from local device  

- Real-time filter adjustments:
  - Brightness
  - Contrast
  - Saturation
  - Hue Rotation
  - Blur
  - Grayscale
  - Sepia
  - Opacity
  - Invert

- Built-in Presets:
  - Normal
  - Vintage
  - Black & White
  - Cool
  - Warm

- Reset all filters  
- Download edited image as PNG  
- Responsive and clean UI  
- Real-time preview using Canvas API  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- HTML5 Canvas API  
- Remix Icons  

---

## 📸 How It Works

1. Upload an image from your device  
2. Adjust filters or choose a preset  
3. Preview changes instantly  
4. Reset if needed  
5. Download the final edited image
